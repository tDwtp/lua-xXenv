Module des56

Provides the DES-based functions.

Usages:

  • require("des56")
  • local des56 = require("des56")

Type des56

des56.crypt(message, key)

Encrypts a string, using DES with a 56-bit key.
des56.decrypt(message, key)

Decrypts a string.

Type des56

Field(s)

des56.crypt(message, key)

Encrypts a string, using DES with a 56-bit key.

Parameters

  • #string message : an arbitrary binary string to be encrypted

  • #string key : an 8-byte binary string to be used as a key

Return value

#string: the cyphertext (binary string)

des56.decrypt(message, key)

Decrypts a string.

The input message must be the result of a previous call to des56.crypt. For any msg and key, we have that 'des56.decrypt(des56.crypt(msg, key), key) == msg'

Parameters

  • #string message : an arbitrary binary cyphertext to be decrypted

  • #string key : key for en-/decryption

Return value

#string: the decrypted text (binary string)
