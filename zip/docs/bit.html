Module bit

C library providing bitwise operations on fixed-sized bitfields implemented as Lua numbers.

Has consistent semantics across 16, 32 and 64 bit platforms as well as IEEE 754 doubles, int32t and int64t lua_Number.

Usages:

  • require("bit")
  • local bit = require("bit")
